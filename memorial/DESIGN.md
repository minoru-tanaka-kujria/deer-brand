# DESIGN.md — 虹の橋メモリアル by Deer

> 亡くなった愛犬の写真からAIが思い出のメモリアル動画を生成するサービス。
> 利益は求めず、運営コスト（API費用＋決済手数料）を除いた全額を動物保護団体に寄付。

---

## 1. Visual Theme & Atmosphere

- **デザイン方針**: 温かみ・静謐・メモリアル。派手さを排し、故犬への敬意と飼い主の想いに寄り添う
- **密度**: ゆったり。余白を多く取り、写真と言葉に集中させる
- **キーワード**: 温かい、静か、誠実、癒し、希望

---

## 2. Color Palette & Roles

Deerブランドカラーをベースに、メモリアルにふさわしい落ち着いたトーンに調整。

### Primary（ブランドカラー）

- **Primary** (`#C9932E`): マスタードゴールド。CTA・アクセントに使用。希望・温かみ
- **Primary Dark** (`#A87A24`): ホバー時

### Semantic（意味的な色）

- **Danger** (`#C0392B`): エラー表示のみ。最小限の使用
- **Warning** (`#D4A017`): 注意
- **Success** (`#6B8E4E`): 完了・成功。Deerグリーン

### Neutral（ニュートラル）

- **Text Primary** (`#2E3544`): ダークネイビー。見出し・本文
- **Text Secondary** (`#6B7280`): 補足テキスト
- **Text Disabled** (`#9CA3AF`): 無効状態
- **Border** (`#E5E0D8`): 温かみのあるボーダー
- **Background** (`#FAF7F2`): アイボリー。Deerのオフホワイト
- **Surface** (`#FFFFFF`): カード背景

### Memorial 専用色

- **Memorial Blue** (`#8BA4B8`): 空・静謐を表現。虹の橋のイメージ
- **Memorial Lavender** (`#B8A9C9`): 安らぎ・癒し。アクセント使用
- **Warm Gray** (`#A89F91`): 写真キャプション・日付

---

## 3. Typography Rules

### 3.1 和文フォント

- **ゴシック体**: Zen Maru Gothic（丸みのある温かいフォント。Deerブランド統一）
- **明朝体**: Noto Serif JP（メモリアルメッセージ・引用に使用）

### 3.2 欧文フォント

- **サンセリフ**: Nunito（ラウンド系。Deerブランド統一）

### 3.3 font-family 指定

```css
/* 見出し・UI */
font-family:
  "Zen Maru Gothic", "Nunito", "Hiragino Maru Gothic ProN", sans-serif;

/* メモリアルメッセージ・引用 */
font-family: "Noto Serif JP", "Hiragino Mincho ProN", serif;
```

### 3.4 文字サイズ・ウェイト階層

| 要素             | size             | weight | line-height | letter-spacing |
| ---------------- | ---------------- | ------ | ----------- | -------------- |
| ヒーロー見出し   | 2.5rem (40px)    | 700    | 1.4         | 0.06em         |
| セクション見出し | 1.75rem (28px)   | 700    | 1.5         | 0.06em         |
| サブ見出し       | 1.25rem (20px)   | 600    | 1.6         | 0.04em         |
| 本文             | 1.0625rem (17px) | 400    | 1.9         | 0.04em         |
| キャプション     | 0.875rem (14px)  | 400    | 1.7         | 0.04em         |
| ボタン           | 1.0625rem (17px) | 600    | 1.0         | 0.08em         |

---

## 4. Layout & Spacing

- **最大幅**: 1080px（読みやすさ重視）
- **セクション間余白**: 100px〜120px
- **カード角丸**: 16px
- **写真角丸**: 50%（円形）または 16px
- **ボタン角丸**: 40px（ピル型）
- **ボタンパディング**: 16px 40px

---

## 5. Components

### ヒーローセクション

- フルワイド背景（グラデーション: アイボリー → 薄いブルー）
- 中央配置テキスト
- メインコピー + サブコピー + CTA
- 下部に犬のシルエットまたは写真（柔らかいフェード）

### ステップカード

- 横3列（モバイル縦積み）
- 番号（ゴールド円）+ アイコン + タイトル + 説明
- 背景白、薄い影

### 寄付セクション

- 左: 寄付の仕組み図解（パイチャートまたはフロー）
- 右: 寄付先団体の紹介
- 背景: 薄いラベンダー

### 料金セクション

- シンプル1プラン表示
- 内訳の透明性（API費用・手数料・寄付額の内訳）

### フッター

- Deerロゴ + 虹の橋メモリアルサブブランド
- リンク + SNS + 運営会社
- 背景: ダークネイビー #2E3544

---

## 6. Animation & Interaction

- **スクロールフェードイン**: 各セクション、0.6s ease-out
- **ホバー**: CTAボタンのみ。控えめなスケール(1.02) + 影の変化
- **パララックス**: 不使用。メモリアルの静けさを保つ
- **ローディング**: スケルトンではなく、中央のシンプルなパルスアニメーション
