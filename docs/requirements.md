## 1. サービス概要

**Deer**は、愛犬の写真をAIでアート作品に変換し、オリジナルグッズとして販売するD2C ECサービス。

### コンセプト

> 愛犬の写真を、世界にひとつだけのアート作品に。

### ターゲットユーザー

- **メインターゲット**: 30〜60代の女性犬オーナー（ファッション意識が高い層）
- **購買力の高い層**: 50〜60代女性（ギフト需要も含む）
- **ペルソナ**: 愛犬との散歩やお出かけを楽しむ、自分だけの特別なグッズを求める女性

## 4. ユーザーフロー

### 4.1 LP導線

```
LP (index.html)
├── ヒーロー → 「無料でつくる」→ オーダーページ
├── アートスタイルセクション → 「全22スタイルを見る」→ スタイル一覧
├── Lookbookセクション（配置バリエーション）
│   ├── 各カード → 配置詳細ページ (placement-detail.html)
│   └── 刺繍カード（Coming Soon / クリック不可）
├── グッズ一覧 → 各カード → オーダーページ
└── お客様の声
```

### 4.2 オーダーフロー（9ステップ）

```
1. 商品選択（パーカー/Tシャツ/トート/マグ/ケース/ポスター/ポストカード）
2. 配置選択（複数配置がある商品のみ）
3. カラー選択（複数色がある商品のみ）
4. サイズ選択（パーカー・Tシャツのみ）
5. 写真アップロード
6. アートスタイル選択
7. プレビュー確認
8. 配送先入力
9. 決済（Stripe: クレジットカード / Apple Pay / Google Pay）
```

### 4.3 配置詳細ページ

```
placement-detail.html?product=XXX&placement=YYY
├── ヒーロー（商品名・配置名・価格）
├── カラー選択バー
├── スタイルサンプル（初期4種: 印象派/水彩/水墨画/ポップアート）
├── 「もっと見る」→ 追加4種（バロック/チャコール/ボタニカル/ワンライン）
└── 各カード → オーダーページへ（パラメータ付き遷移）
```

## 5. 技術構成

### 5.1 フロントエンド

- **構成**: 静的HTML/CSS/JS（フレームワーク未使用）
- **ホスティング**: Vercel
- **フォント**: Cormorant Garamond（serif）+ Zen Maru Gothic（sans）
- **カラーパレット**:
  - Ivory: `#FAF7F2` / Cream: `#F5EDE3`
  - Gold: `#C4A265` / Terracotta: `#C4836A`
  - Charcoal: `#2C2824`
- **アニメーション**: IntersectionObserverによるスクロール連動fade-up

### 5.2 決済

- **Stripe**（クレジットカード / Apple Pay / Google Pay）
- 現時点はフロントエンドのみ（バックエンド未実装）

### 5.3 AI画像変換

- **OpenRouter API** 経由で Gemini 2.5 Flash を使用
- 画像生成スクリプト: `~/.claude/bin/image-gen.sh`

### 5.4 リポジトリ

- **GitHub**: tanaka-team/products（mainブランチ）
- **本番URL**: https://deer-brand.vercel.app
