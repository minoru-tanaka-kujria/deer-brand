Deer Brandとは

Deer Brandは、愛犬の写真を1枚アップロードするだけで、AIが23種類のアートスタイルに変換し、パーカー・Tシャツ・マグカップなどのオリジナルグッズをお届けするサービスです。

特長 内容
アートスタイル 23種類（印象派・水彩・ポップアートなど）
送料 全品送料無料
お届け日数 最短5日
決済 Stripe（クレジットカード・各種決済）
ギフト対応 ギフトラッピング対応

アートスタイルの選び方

上部のカテゴリタブで絞り込みができます。

水彩画 にじみ水彩、ボタニカル水彩、ウェットオンウェット

⑨確認
ステップ①：商品選択 — モデル着用・使用イメージのカードから商品を選びます
  1. ① 商品選択 一覧カードからお気に入りの商品×スタイルの組み合わせをクリックして選択します。パーカー・Tシャツ・マグカップ・スマホケース・ポスターなど多数あります。
  2. ② プリント配置の選択 左胸ワンポイント / 胸中央プリント / バックプリント（+¥2,000）から選択します。
  3. ③ カラーの選択 商品カラーを選択します（グレー・ネイビー・ブラック・ピンク・ホワイトなど）。
  4. ④ サイズの選択 S / M / L / XL / 2XL から選択します。
  5. ⑤ 愛犬の写真アップロード 愛犬の写真を1枚アップロードします。正面からの写真が推奨です。対応形式：JPEG / PNG（最大10MB）。
  6. ⑥ AIプレビューの確認 AIがアートスタイルに変換したプレビュー画像が表示されます。満足したら次に進みます。気に入らない場合は「再生成」ボタンで別の結果を試せます。
  7. ⑦ お届け先の入力 氏名・郵便番号・住所・電話番号を入力します。ギフトラッピング希望の場合はチェックを入れます。
  8. ⑧ お支払い情報の入力 Stripe決済フォームでクレジットカード情報を入力します。クーポンコードがある場合はここで適用できます。
  9. ⑨ 注文確認・完了 注文内容を確認して「注文を確定する」をクリックします。注文完了後、メールで確認メールが届きます。注文IDは DEER-YYYYMMDD-XXXX 形式で発行されます。
商品一覧（下部）— マグカップ・スマホケース・ポスター・ボストカード・刺繍キャップ・ステッカーなど
写真アップロードのコツ 愛犬の正面を向いた写真、明るい場所で撮影した写真がアートに変換した際に最も美しく仕上がります。

マイページで注文確認

マイページ（https://deer-brand.vercel.app/account）— ログインが必要です

マイページではログイン後に以下が確認できます。

機能 内容
注文履歴 過去の注文一覧（注文ID・商品・金額・ステータス）
配送状況 現在の注文ステータス（受付済み / 制作中 / 発送済み / 完了）
アカウント設定 メールアドレス・パスワードの変更
  1. ログイン方法 「ログイン / 新規登録」ボタンをクリックしてモーダルを開きます。
  2. 認証方法の選択 メールアドレス＋パスワード、またはGoogleアカウントでログインできます。
  3. 注文履歴の確認 ログイン後、注文一覧が表示されます。注文IDをクリックすると詳細が確認できます。
注文なしで購入した場合 ゲストとして購入した場合、注文確認メールのみで追跡できます。マイページで確認するにはアカウント作成が必要です。
